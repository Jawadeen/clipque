TikTok authorization callback

This page is a fallback landing page for ClipQue's TikTok connection flow. The actual token exchange happens server-side at /.netlify/functions/tiktok-callback, which then hands control back to the ClipQue desktop app automatically. You should only see this page if that automatic handoff couldn't reach the app.

Checking authorization result…

Authorization error:

Go back to ClipQue and try connecting your TikTok account again. If the desktop app wasn't open when this happened, open it first, then retry.

Authorization completed. If ClipQue did not automatically detect this, return to the app and click "Connect TikTok" again to retry locally.
